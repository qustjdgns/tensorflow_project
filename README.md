****League of Legends Contribution Model (LoL 기여도 분석 모델) 최종 보고서****

---

**1. 프로젝트 개요**

이 프로젝트는 리그 오브 레전드(LoL) 경기 데이터를 분석하여, 단순한 KDA나 딜량 통계의 한계를 넘어 포지션별 역할과 경기 시간대별 흐름을 반영한 객관적인 기여도 평가 모델을 구축하는 것을 목표로 합니다.

핵심 접근법:
중앙값(Median) 기반 상대 평가: "남들보다 얼마나 잘했는가?"를 측정하여 챔피언/메타 유불리 최소화.
스냅샷(Snapshot) 기반 타임라인 평가: 승패가 갈리는 주요 시간대(8~20분) 집중 분석.
역할군별 가중치 적용: 딜러, 탱커, 서포터의 서로 다른 기여 방식을 수치화.

---
*** 2. 데이터 전처리 및 피처 정의 ***
본 모델은 경기 데이터를 Timeline(분당 흐름)과 Final(최종 성과) 두 가지 차원에서 분석합니다.

2-1. Timeline Features (t_ 접두사)
게임의 성장 과정을 평가하기 위해 1분 단위로 수집된 데이터입니다. 8/10/12/15/20분 스냅샷 평가의 기초가 됩니다.

| 피처명 / 설명 / 주요 평가 대상 |  |  |
|-------------------------------|-----------------------------|------------------------------|
| t_totalGold | 분당 누적 골드 획득량 | 전 라인 (경제력) |
| t_xp | 분당 누적 경험치 | 전 라인 (레벨링) |
| t_damageToChampions | 챔피언에게 가한 누적 피해량 | 딜러 라인 (TOP, MID, ADC) |
| t_minionsKilled | 처치한 미니언 수 (CS) | 라이너 (TOP, MID, ADC) |
| t_jungleMinionsKilled | 처치한 정글 몬스터 수 | 정글러 (JUNGLE) |


2-2. Final Stats Features (f_ 접두사)
게임 종료 시점의 최종 성과 지표로, 5-4(시간대별 곡선)의 하이브리드 점수 산출 시 30% 비중으로 반영됩니다.

| 피처명 / 설명 / 핵심 반영 라인 |  |  |
|-------------------------------|-----------------------------|-----------------------------|
| f_killParticipation | 킬 관여율(KP) | JUNGLE, MID, SUP |
| f_visionScore | 시야 점수 | SUP, JUNGLE |
| f_soloKills | 솔로 킬 횟수 | TOP, MID (라인전 기량) |
| f_damageDealtToTurrets | 포탑에 가한 피해량 | TOP, ADC (운영/철거) |

*** 3. 기여도 산출 로직 및 가중치 ***
각 포지션의 역할이 다르므로, 기여도 산출 시 라인별로 차별화된 가중치를 적용했습니다.

3-1. Timeline Score 가중치 (기본 기여도)
8, 10, 12, 15, 20분 스냅샷에서 이 가중치를 적용하여 점수를 산출합니다.

| 라인(Role) / Rel. Gold / Rel. XP / Rel. Damage / Rel. Lane CS / Rel. Jungle CS / 비고 |  |  |  |  |  |  |
|-----------------------------------------------------------------------------------------|--------|--------|----------|-----------|--------------|------------------------------|
| TOP / MID / ADC | 0.3 | 0.2 | 0.3 | 0.2 | - | 성장과 딜링 밸런스 중시 |
| JUNGLE | 0.3 | 0.3 | 0.1 | - | 0.3 | 레벨링과 정글링 효율 중시 |
| SUP | 0.4 | 0.4 | 0.2 | - | - | 골드·XP 수급 난이도 고려 |

가중치 설계 근거: Timeline 피처 간의 내적 상관계수는 평균 0.42 수준입니다. 각 피처가 '성장'이라는 공통 목적을 가지면서도 서로 독립적인 정보를 담고 있어 분산 가중치를 적용했습니다.

3-2. Final Stats Score 가중치 (하이브리드용)
오직 5-4(시간대별 곡선) 분석에서만 사용되며, Timeline(0.7) + Final(0.3) 공식으로 결합됩니다.

| 라인 / 제1요소 (40~50%) / 제2요소 (30~40%) / 제3요소 (10~20%) / 제4요소 (10%) |  |  |  |  |
|----------------------------------------------------------------------------------|------------------|------------------|------------------|----------------|
| TOP | 솔로킬 0.4 | 타워딜 0.4 | KP 0.1 | 시야 0.1 |
| JUNGLE | KP 0.4 | 시야 0.4 | 오브젝트 스틸 0.2 | - |
| MID | KP 0.5 | 솔로킬 0.3 | 타워딜 0.1 | 시야 0.1 |
| ADC | KP 0.5 | 타워딜 0.4 | 시야 0.1 | - |
| SUP | 시야 0.4 | KP 0.2 | 힐량 0.2 | CC기 0.2 |

*** 4. 스냅샷(Snapshot) 기반 평가 로직 ***
기여도는 전 구간 단순 평균이 아닌, 승패가 갈리는 핵심 시간대(Critical Moments)의 데이터를 추출하여 계산합니다.

스냅샷 시점: 8분, 10분, 12분, 15분, 20분

선정 이유:  
8~10분: 라인전 주도권 결정 및 첫 전령 교전.  
12~15분: 포탑 방패 골드 채굴 종료 및 라인전 페이즈 종료.  
20분: 바론 생성 및 중반 운영(사이드 스플릿/한타) 전환점.

산출 방식: 위 5개 시점의 Timeline Score 평균값을 해당 선수의 최종 기여도로 정의합니다.

*** 5. 분석 결과 및 시각화 ***
5-1. 라인별 종합 기여도 분포 (승리 vs 패배)  
데이터: 8~20분 스냅샷 평균 기여도.

해석:  
승리팀: 기여도 중앙값이 1.1 내외에 위치하며, 상위권으로 꼬리가 깁니다. 이는 승리팀이 주요 시간대에 상대보다 더 많은 자원을 획득했음을 보여줍니다.  
패배팀: 1.0 미만 구간에 대다수가 분포합니다.  
결론: 스냅샷 기반 기여도 모델이 승패를 가르는 '체급 차이'를 명확히 설명합니다.

5-2. 개인별 일관성 분석 (Bonnie#0314)  
지표 해석:  
내 평균: 1.01 (전체 평균 1.0과 일치).  
평가: 승리/패배 여부와 관계없이 본인의 성장 지표는 평균(1.0)을 꾸준히 방어하는 안정적인 플레이어입니다.

5-3. 라인별 소환사 유형 군집 분석 (K-Means)  
BOTTOM 라인 해석:  
기여도가 높으면서(>1.1) 표준편차(기복)가 낮은 '안정적 캐리형' 그룹이 명확히 존재합니다. 바텀 라인전 승리가 게임 전체 승률에 큰 영향을 미침을 알 수 있습니다.  
MID 라인 해석:  
'하이 리스크 하이 리턴' 성향이 강함을 보여줍니다.

5-4. Match ID 367 시간대별 기여도 (승리 vs 패배)  
경기 흐름 분석:  
승리팀 바텀: 게임 시작부터 끝까지 기여도 1.4~1.6 유지하며 상대를 압도했습니다.  
패배팀 바텀: 0.8 아래로 무너졌습니다.  
미드: 초반 평범했으나 바텀 우세를 기반으로 20분경 1.3까지 성장했습니다.  
결론: 바텀의 라인전 우위가 미드의 성장 공간을 형성한 게임입니다.

** 6. 모델 성능 검증 (Model Validation) **
6-1. 변별력 검증 (ROC Curve Analysis)  
AUC Score: 0.593  
해석: 개인의 8~20분 지표만으로 승패를 약 60% 설명할 수 있습니다. 5:5 팀 게임에서 개인 지표가 이 정도 설명력을 가진다는 것은 개인의 라인전 체급이 승률에 유의미한 영향을 미친다는 것을 의미합니다.

*** 6-2. 예측 정확도 검증 (Confusion Matrix) ***
정확한 분류: 캐리(TP)와 역적(TN)을 잘 구분했습니다.  
흥미로운 오분류(FP 2188건): 기여도는 높았으나 패배한 경우입니다. '졌잘싸' 유형으로, 본인의 지표는 훌륭했으나 팀 운이 따르지 않은 케이스를 의미합니다.
